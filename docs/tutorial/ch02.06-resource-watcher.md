## 2.6 Implement the resource watcher

[<< Previous](ch02.05-kube-client.md) | [Contents](ch02.00-kubernetes-core.md) | [Next >>](ch02.07-resource-list-state.md)

---

The watcher runs in a background tokio task and sends pod snapshots through a
bounded channel. It uses kube-rs's built-in watcher which handles reconnection,
relisting, and error recovery automatically.

**crates/crystal-core/src/informer.rs:**

```rust
use std::collections::HashMap;

use futures::StreamExt;
use k8s_openapi::api::core::v1::Pod;
// kube 3.x watcher types — Event has 5 variants, not the old 3.
use kube::runtime::watcher::{self, Event};
use kube::Api;
// Bounded channel — prevents unbounded memory growth if the UI is slow.
use tokio::sync::mpsc;
// CancellationToken provides cooperative cancellation for async tasks.
// When cancelled, the background watcher exits its loop cleanly.
use tokio_util::sync::CancellationToken;
use tracing::{info, warn};

use crate::resource::PodSummary;

// Events sent from the watcher to the UI event loop.
// Updated contains a full snapshot (not a delta) — this simplifies the consumer
// because it can just replace its entire state rather than merging changes.
#[derive(Debug, Clone)]
pub enum ResourceEvent<S> {
    Updated(Vec<S>),
    Error(String),
}
```

### The ResourceWatcher struct

```rust
pub struct ResourceWatcher {
    cancel: CancellationToken,
}

impl ResourceWatcher {
    // Start watching pods from the given API handle.
    // Returns a ResourceWatcher that can be dropped to stop the background task.
    //
    // The tx channel has capacity 16 — if the UI isn't draining fast enough,
    // updates are dropped. This is fine because each update is a full snapshot;
    // the UI will get the latest state when it catches up.
    pub fn watch_pods(api: Api<Pod>, tx: mpsc::Sender<ResourceEvent<PodSummary>>) -> Self {
        let cancel = CancellationToken::new();
        let cancel_clone = cancel.clone();

        tokio::spawn(async move {
            // watcher::watcher returns a Stream of Result<Event<Pod>, watcher::Error>.
            // It handles reconnection and relisting automatically using exponential backoff.
            let stream = watcher::watcher(api, watcher::Config::default());
            // pin! is required to use a Stream in a loop — it ensures the stream
            // is not moved in memory between polls (a Rust async requirement).
            tokio::pin!(stream);

            // Local accumulator: we maintain a HashMap of all known pods.
            // On each event, we update the map and send a full snapshot.
            let mut pods: HashMap<String, PodSummary> = HashMap::new();

            loop {
                // select! races two futures: cancellation and the next stream item.
                // If the token is cancelled (e.g., watcher is dropped), we exit cleanly.
                tokio::select! {
                    _ = cancel_clone.cancelled() => {
                        info!("Pod watcher cancelled");
                        break;
                    }
                    item = stream.next() => {
                        match item {
                            Some(Ok(event)) => {
                                // kube 3.x Event variants:
                                //   Apply     — a pod was created or modified
                                //   InitApply — a pod arrived during initial list
                                //   Delete    — a pod was removed
                                //   Init      — the watcher is restarting (clear cache)
                                //   InitDone  — initial listing complete
                                match event {
                                    Event::Apply(pod) | Event::InitApply(pod) => {
                                        let summary = PodSummary::from(&pod);
                                        let key = format!("{}/{}", summary.namespace, summary.name);
                                        pods.insert(key, summary);
                                    }
                                    Event::Delete(pod) => {
                                        let name = pod.metadata.name.as_deref().unwrap_or_default();
                                        let ns = pod.metadata.namespace.as_deref().unwrap_or("default");
                                        let key = format!("{ns}/{name}");
                                        pods.remove(&key);
                                    }
                                    Event::Init => {
                                        pods.clear();
                                    }
                                    Event::InitDone => {}
                                }
                                let snapshot: Vec<PodSummary> = pods.values().cloned().collect();
                                // If the channel is full, the send returns Err and we drop the update.
                                // The next successful send will contain the up-to-date snapshot.
                                let _ = tx.send(ResourceEvent::Updated(snapshot)).await;
                            }
                            Some(Err(e)) => {
                                warn!("Watcher error: {e}");
                                let _ = tx.send(ResourceEvent::Error(e.to_string())).await;
                            }
                            None => break, // stream ended
                        }
                    }
                }
            }
        });

        Self { cancel }
    }

    pub fn stop(&self) {
        self.cancel.cancel();
    }
}

// Drop impl ensures the background task is cancelled when the watcher is dropped.
// Without this, dropping the watcher would leave an orphaned task running forever.
impl Drop for ResourceWatcher {
    fn drop(&mut self) {
        self.cancel.cancel();
    }
}
```
